CLDF v4.7.10 – DIREKT AUF GITHUB HOCHLADEN

1. ZIP vollständig entpacken.
2. GitHub-Repository öffnen.
3. Add file → Upload files wählen.
4. Den Inhalt des entpackten Ordners hochladen. Die Dateien müssen direkt im Repository-Stamm liegen.
5. Vorhandene Dateien ersetzen und Commit changes bestätigen.
6. Warten, bis die GitHub-Pages-Aktion abgeschlossen ist.
7. App auf dem Handy vollständig schließen und erneut öffnen.

Die Fingerprints sind bereits auf mehrere Dateien unter 25 MB verteilt. Nutzer müssen keine Referenzen importieren.
