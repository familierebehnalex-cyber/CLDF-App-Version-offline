# Video- und Bewegungsanalyse

Die Videoanalyse ist eine lokale Zusatzfunktion. Sie kann Bewegungsintensität und zeitliche Muster aus einer vom Benutzer ausgewählten Aufnahme auswerten. Sie ersetzt keine sichere Identifikation einer Choreografie und wird niemals vor eine direkte Lied–Tanz-Zuordnung oder einen sicheren Audio-Fingerprint-Treffer gesetzt.

Reihenfolge der Empfehlung:

1. sicherer lokaler Audio-Fingerprint
2. feste Lied–Tanz-Zuordnung
3. BPM, Taktart, Motion und Rhythmus
4. optionale Video-Bewegungsmerkmale

Die Videodatei wird lokal verarbeitet und nicht hochgeladen.
