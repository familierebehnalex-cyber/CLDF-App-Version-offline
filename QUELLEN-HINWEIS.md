# Technische Grundlage

Das Projekt folgt der offiziellen ShazamKit-Android-Dokumentation von Apple,
Version 2.1.1. ShazamKit wird als AAR eingebunden. Für den Shazam-Katalog sind
Internet und ein DeveloperTokenProvider erforderlich. Streaming-Audio muss PCM
16 Bit, Mono und 48/44,1/32/16 kHz sein; dieses Projekt verwendet 48 kHz.

Offizielle Einstiegsseite:
https://developer.apple.com/shazamkit/
