# CLDF Offline-App v4.7.7

Vollständige GitHub-Pages-Version mit fest eingebauten Audio-Fingerprints.

## Enthalten

- 3480 bereinigte Fingerprints für 2323 Lieder
- 4.011 Song-API-Metadatensätze
- lokale Lied-Erkennung ohne laufende Kosten
- automatische Lied–Tanz-Zuordnung und gekennzeichnete Tanzvorschläge
- Offline-Cache und GitHub-Pages-Workflow

## Hochladen

Den gesamten Inhalt dieses Pakets direkt in den Stamm des GitHub-Repositorys hochladen und vorhandene Dateien ersetzen. Danach die veröffentlichte App vollständig schließen und neu öffnen. Bei einer installierten PWA kann zusätzlich einmal der Browser-Cache beziehungsweise die alte App-Installation entfernt werden.
